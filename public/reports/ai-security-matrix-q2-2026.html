Executive Summary

The AI security market underwent rapid consolidation in Q1 2026. Palo Alto Networks entered acquisition talks with Protect AI (valued at $400M), signalling that AI Security Posture Management (AI-SPM) is no longer a niche — it's a boardroom priority.

Meanwhile, prompt injection attacks have evolved from proof-of-concept to production exploit. Lakera's Firewall reported blocking 2.3 million injection attempts in Q1 alone. Shadow AI usage — employees connecting unsanctioned LLMs to internal data — has emerged as the dominant governance challenge for enterprise CISOs.

This matrix evaluates 6 purpose-built solutions across four capability areas: LLM Firewalls, AI-SPM, GenAI Governance, and LLM Red-Teaming — giving security teams a clear lens for vendor selection.

Vendor Profiles

What Makes Each Vendor Unique

Brief breakdowns to help your team shortlist vendors based on your threat model and deployment requirements.

Lakera

LLM Firewall
prompt injection · data leakage · content safety

Lakera's Guard sits as an API gateway in front of any LLM — OpenAI, Anthropic, self-hosted models. It inspects every prompt and completion for injection patterns, PII leakage, and harmful content in real-time. The free tier makes it the lowest-friction entry point for teams shipping their first LLM feature. Deploying takes minutes, not sprints.

Free tier available lakera.ai →

HiddenLayer

Enterprise
model defence · adversarial ML · runtime monitoring

Backed by $56M+ in funding, HiddenLayer's MLDR (ML Detection and Response) platform monitors deployed models for adversarial manipulation, model theft via side channels, and supply chain compromise. Think of it as EDR for your ML models. Best suited for organisations running ML inference at scale — financial services, healthcare, defence.

Enterprise pricing hiddenlayer.com →

Protect AI

AI-SPM
ML supply chain · model scanning · posture management

Protect AI scans your entire ML pipeline — from training data to model artifacts to deployment configs — for vulnerabilities. Their Guardian product catches poisoned models, unsafe serialisation, and misconfigured access controls. Valued at $400M with Palo Alto Networks acquisition interest, this is the vendor defining AI Security Posture Management.

Enterprise pricing protectai.com →

Prompt Security

GenAI Governance
shadow AI · prompt DLP · compliance enforcement

The governance layer for enterprise GenAI adoption. Prompt Security gives CISOs visibility into which LLMs employees are using, what data they're feeding in, and whether those interactions comply with corporate policy. The "shadow AI" problem — employees using ChatGPT with customer data — is the #1 governance challenge in 2026. Prompt Security addresses it directly.

Enterprise pricing prompt.security →

Rebuff

Open Source
prompt injection · multi-layer · self-hostable

An open-source framework (maintained by the Protect AI team) for detecting prompt injection using three layers: heuristic analysis, LLM-based classification, and vector similarity matching against a database of known attack patterns. Ideal for teams who want to build prompt defence into their stack without vendor lock-in.

Free (Apache 2.0) GitHub →

Garak (NVIDIA)

Open Source
LLM red-teaming · vuln scanning · CI integration

The "nmap for LLMs." NVIDIA's Garak probes language models for hallucination vulnerabilities, prompt injection susceptibility, data extraction risks, and jailbreak vectors. Runs from the CLI or integrates into CI pipelines. Essential for red teams assessing LLM deployments before production launch.

Free (Apache 2.0) GitHub →

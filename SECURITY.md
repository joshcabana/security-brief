# Security Policy

## Supported Versions

Currently, only the latest release on the `main` branch is supported for security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not disclose it publicly or open a public issue.

Instead, please send a direct message or email to the repository owner providing details of the issue, the environment in which it was discovered, and steps to reproduce.

You should receive an initial acknowledgment within 48 hours. Valid vulnerabilities will be addressed, tested, and pushed directly to `main`.
